# To Run
1.  Open command line cmd at the root of the repository.
2.  Run the command   

        `python main.py`.
3. Follow the commands in the menu to add/delete further nodes.

        1. Add More Nodes  : 'a'
        2. Delete Some Nodes : 'd'
        3. Exit or Do nothing : 'e'

4. Connected Node Graph opens instantly.
